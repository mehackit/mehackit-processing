Welcome

This website contains short examples and exercises to help you getting started with programming. The aim is to explore what kind of visual things you can make with code and a bit on how to achieve those things. We'll be working with a language and environment called Processing, which is widely used by students, artists, designers, researchers, and hobbyists. It's a good choice for for learning how to code within the context of the visual arts.

First, download and install Processing. After that, press right arrow to move to the next slide. Next slide explains how to use this site.

Introduction

The material is divided into four sections. Each section exhibits a small project and introduces new concepts along the way. After completing a section you can start creating your own project or move to the next section. Each section gives you more inspiration and tools to work with.

The goal is that you create your own work of art and export it to a video. You can first follow the first section to get started and then decide wereither to move to the next section or start working on your own project.

Rectangles and shapes

Let's start with drawing basic shapes to the canvas. You can open Processing and create new sketch or try the examples with the editor below. Notice that the browser editor doesn't save your work and Processing editor might help you if you make mistakes. Below you find some heavily commented Processing code. The example walks you through the basics of a processing sketch. The statements that make up the program are divided into two functions: setup() and draw(). Statements in setup function are executed once when the program begins and statements in the draw function are executed again and again, as long as the program runs.

In the draw function you find the following statement: 'rect(100, 100, 50, 50);'. This statement tells the computer to draw a rectangle with the x-coordinate 100, y-coordinate 100, width of 50 and height of 50. Try to change the paremeters, so that the rectangle is drawn with coordintaes(200,200) and width and height of 112. After that try to draw circles for eyes with the ellipse function and line as a mouth with the line function.

Variables


        
Quite often you'll want to store something. You can store different things with a thing called variable. A variable is like a box where you can put things and take things out when you need them.

        
In Processing you'll have to state what kind of things you want to store in the box. For a computer a whole number (1, 2, 4902, ...) is a different thing than a decimal number (1.0, 3.14, ...). So when you create a variable, you'll have to state is the variable for a whole number, decimal number, a string or maybe something other (boolean, color, int, float, String are quite common, but for now just use float).

        
It also matter where you create your variable. If you create your variable inside the draw function, it will only be visible in the draw function. If you create the variable before setup funciton, it will be visible to the whole sketch.

Colors, fill, stroke and background


        
So far, everything has been quite grey. Now it's the time to add some color. Color tutorial by Daniel Shiffman is a good place to start. Take a look at it and then come back.

        
Fill function changes the fill color of all drawn things until another fill function is called. The same applies to stroke, but stroke changes the outline color. Background paints the whole screen with the given color.

        
Try to add some color to your previous animation. You can use random function with colors too if you wish.

In the example below, I have used a mouseMoved function. This function gets called (and its statements executed) every time the mouse is moved. In the draw function I have used an if-statement. If-statement checks if the given condition is true and, if it is, executes the statements
